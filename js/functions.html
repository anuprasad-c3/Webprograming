JavaScript Predefined Functions: String & Math Objects

1. String Functions

String: " Hello JavaScript! "

  • trim() - Removes whitespace from both ends
  • toUpperCase() - Converts to uppercase
  • toLowerCase() - Converts to lowercase
  • indexOf("Java") - Finds the index of a substring
  • slice(1, 6) - Extracts part of the string
  • length - Returns string length
  • charAt(6) - Return character at position 6
  • concat(" is awesome!") - Joins strings
  • replace('JavaScript', 'World') - Replaces text
